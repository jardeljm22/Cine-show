## 🧠 💡 Sobre o projeto

O CineShow foi desenvolvido com o objetivo de simular uma aplicação real de catálogo  e exibição de filmes, focando em:

* consumo de API externa
* organização de componentes em React
* construção de interface moderna
* boas práticas de desenvolvimento front-end

---

## ✨ Funcionalidades

* 🎬 Listagem de filmes populares
* 📄 Página de detalhes de cada filme
* ⭐ Exibição de avaliação dos usuários
* 🖼️ Imagens dinâmicas da API
* 🔄 Navegação entre páginas
* 📱 Layout responsivo (em evolução)
*    exibição de filmes

---

## 🛠️ Tecnologias utilizadas

* React

* JavaScript (ES6+)
* CSS3

## ⚙️ Como rodar localmente

### 📥 Clone o projeto

```bash
git clone https://github.com/jardeljm22/Cine-show.git
```

---

### 📂 Acesse a pasta

```bash
cd Cine-show
```

---

### 📦 Instale as dependências

```bash
npm install
```

---

### 🔑 Configure as variáveis de ambiente

Crie um arquivo `.env`:

```env
VITE_TMDB_API_KEY=your_api_key
VITE_TMDB_BASE_URL=https://api.themoviedb.org/3/
VITE_TMDB_URL_IMAGE=https://image.tmdb.org/t/p/w500
```

---

### ▶️ Execute o projeto

```bash
npm run dev
```
